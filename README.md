# simple text Editor

A lightweight text editor built with C# and Windows Forms.

## Features
- Open, Save, Save As
- Undo, Copy, Cut, Paste
- Keyboard shortcuts (Ctrl+S, Ctrl+O, etc.)
- Portable single-file EXE
- Custom application icon

## Download
Download the latest version from the Releases page:
https://github.com/mepavankumar15/Avyu-textEditor/releases

## How to Run
Simply download `MyTextEditor.exe` and double-click it. No installation required.
